# Qualitätssicherung durch Human-in-the-Loop und Kontrollschichten

KI kann Fehler machen, deshalb gehört an kritische Stellen eine menschliche Freigabe. Üblich ist ein Human-in-the-Loop, bei dem der Ablauf vor einer wirksamen Aktion anhält und eine Person per Chat oder Nachricht zustimmt oder einen neuen Entwurf anfordert. Zusätzlich lassen sich mehrere Agentenschichten kombinieren, sodass ein Agent die Ergebnisse eines anderen prüft, ergänzt um automatische Plausibilitätskontrollen. So entsteht ein Vier-Augen-Prinzip, das unkontrollierte Fehler in Produktivsystemen verhindert.
